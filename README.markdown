![GitHub Stars](https://img.shields.io/github/stars/shariarmahmudsaif/stylesync?style=social)
![GitHub Issues](https://img.shields.io/github/issues/shariarmahmudsaif/stylesync)

StyleSync is an open-source collection of modern, clean UI components for web developers. It features a refined homepage with a clear intro, responsive design, and 50 interactive div and section effects, along with loaders, fonts, modals, inputs, buttons, and text styles.

## Features

- **Clean Homepage**: Minimalistic UI with a prominent intro, typewriter effect, liquid fill buttons, and a responsive burger menu.
- **50 Div & Section Effects**: Elegant effects like Glass Glow Panel, Gradient Overlay, and Starburst Hover, built with CSS and JavaScript.
- **Responsive Design**: Adapts across devices (mobile, tablet, desktop) with Tailwind CSS.
- **Accessibility**: ARIA labels, keyboard navigation, and WCAG-compliant contrast.
- **Typography**: Poppins for body text, Playfair Display for headings.
- **Non-Overlapping Sections**: Clear separation with generous spacing (4rem desktop, 2rem mobile).
- **Modular Structure**: Easily integrate components into your projects.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shariarmahmudsaif/stylesync.git
   ```
2. Navigate to the project directory:
   ```bash
   cd stylesync
   ```
3. Serve locally:
   ```bash
   python -m http.server 8000
   ```
   Visit `http://localhost:8000` to view the homepage.

## Usage

- **Homepage**: Explore `index.html` for a clean UI with div effect previews and navigation.
- **Div Effects**: Visit `div-effects/index.html` for all 50 effects with code snippets.
- **Other Components**: Access loaders, fonts, modals, inputs, buttons, and text styles via navigation links.
- **Customization**: Copy CSS/JS from `div-effects` snippets to integrate components.
